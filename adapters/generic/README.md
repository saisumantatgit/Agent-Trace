# Agent-Trace — Generic Prompts

For LLMs without native plugin support, use the prompts in `prompts/` directly:

1. **Before risky edits**: Paste `prompts/trace.md` into your conversation
2. **To build repo universe**: Paste `prompts/map.md`
3. **To query impact**: Paste `prompts/query.md`
4. **To validate**: Paste `prompts/validate.md`

These are the LLM-agnostic core prompts that all platform-specific adapters wrap.
